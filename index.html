らびぃーゆーのお部屋

自己紹介



自作PC・ガジェット好きの専門学生です。ゲームはミリシタやR6Sをやっています。
特に好きなキャラはミリオンライブの北上麗花さんです（声がめっちゃすこ）。
今のところC言語がほんの少しだけかけます。
かなり前にBASICを触っていましたがほとんど忘れました。
ちなみに「らびぃーゆー」というハンドルネームの由来は
初めに使ったノートPCがPC-LU50L54DC(Lavie U)だからです。
最近うれしかったことはWIMAX2+を卒業し光回線を開通できたことです。
これからしてみたいことは電子工作です。

デバイス

所持デバイスの一部を紹介！

メインデスク

Ryzen7-2700XとRTX2060をMini-ITXケース「InWin A1」に詰め込んだコンパクトゲーミングPCです。
ストレージはOS用とデータ用で2つのSSDを搭載しています。

メインノート

2016年に購入したCorei7-6700HQ搭載の大画面ノートPCです。
ストレージをSSDに換装、メモリを16GBに増やして使っています。
また、WLANを交換してWIFI6(ax)に対応させました。

サーバー

以前に組んでいた自作PCのパーツを主に流用したサーバー機です。
最近、Windows Server 2019からUbuntu Server 18.04LTSに乗り換えました。

スマホ

AQUOS Rから機種変してAQUOS R3になりました。
正直、電池持ちが前よりよくなかったりと個人的に使い勝手が悪くなった印象が強い機種です。（AQUOS Rと比較して）

タブレット

学習用と言い訳をつけて購入したタブレットですが、ApplePencilの書き心地の良さの影響かノートをとることにしか使わなくなりつつあります。
たまにこれでミリシタとかやってます。
